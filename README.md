# Timer Light
Responsável por controlar a iluminação do aquário e disponibilizar controle remoto que permite a manipulação pela internet.
Geração de logs no slack.

Spring boot - raspberry pi 3 - slack

Requisitos:

Maven: 3.3.*
Java: 8
Raspberry Pi: 3

Comece com:

mvn clean package
mvn spring-boot:run
